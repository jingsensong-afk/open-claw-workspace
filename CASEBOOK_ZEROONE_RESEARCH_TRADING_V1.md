# 案例 1：美伊战争 / 地缘冲突升级

## 1. 事件
- 中东地缘冲突升级，美伊战争预期升温

## 2. 核心判断
- 风险偏好下降，避险情绪抬升
- 资金优先流向黄金、原油、军工、战争相关预测市场
- 风险资产（美股高 Beta、BTC）短期承压

## 3. 逻辑链
地缘冲突升级 → 风险偏好下降 → 避险资产受追捧 → 战争受益链走强 → 风险资产被减仓

## 4. 资金流向
### 流入
- 黄金
- 原油
- 军工
- 战争相关预测市场
- 美元 / 美债

### 流出
- 美股高 Beta
- BTC / ETH 等风险资产

## 5. 跨市场映射标的
### 加密
- BTC
- ETH
- 高 Beta 山寨（回避）

### 港美股
- LMT
- RTX
- NOC
- XLE
- QQQ / SPY（减仓观察）

### 大宗商品
- XAUUSD
- WTI
- Brent

### 预测市场
- 停火 / 战争结束相关 opposite leg
- 战争升级 / 地缘冲突持续合约

## 6. 主交易标的
- XAUUSD
- WTI
- Brent

## 7. 高弹性标的
- LMT
- RTX
- NOC

## 8. 观察标的
- BTC
- ETH
- QQQ
- SPY
- 美元指数 DXY
- 美债 ETF TLT

## 9. 触发条件
- 冲突进一步升级
- 官方/媒体持续释放升级信号
- 黄金、原油同步放量上行

## 10. 失效条件
- 停火谈判出现
- 冲突明显降级
- 原油与黄金冲高回落

## 11. 风险提示
- 地缘消息极易反复
- 标的波动大，容易出现反复插针
- 需区分“风险溢价”与“基本面趋势”

## 12. 事后验证
- 后续记录：谁先涨、谁滞后、哪些映射最有效、哪些是假信号

# 案例 2：特朗普称战争将很快结束

## 1. 事件
- 特朗普公开表态：战争将很快结束

## 2. 核心判断
- 这是“冲突降级预期”的前置信号
- 若市场相信，则战争溢价回落，风险偏好修复

## 3. 逻辑链
表态 → 市场预期战争降级 → 黄金/原油战争溢价回吐 → 风险资产修复

## 4. 资金流向
### 流出
- 黄金
- 原油（战争溢价部分）
- 军工高弹性标的

### 流入
- BTC / ETH
- QQQ / 科技成长
- 港股科技

## 5. 跨市场映射标的
### 加密
- BTC
- ETH

### 港美股
- QQQ
- SPY
- NVDA
- AMD
- TSLA
- LMT / RTX / XLE（相对回避）

### 大宗商品
- XAUUSD
- WTI
- Brent

### 预测市场
- 停火 / 战争结束概率相关合约

## 6. 主交易标的
- BTC
- ETH
- QQQ
- XAUUSD
- WTI

## 7. 高弹性标的
- TSLA
- AMD

## 8. 观察标的
- TLT
- DXY
- LMT
- XLE

## 9. 触发条件
- 后续出现谈判 / 停火 / 官方缓和信号
- 原油转弱
- BTC / 科技同步修复

## 10. 失效条件
- 冲突继续升级
- 黄金、原油重新转强
- 风险资产反弹失败

## 11. 风险提示
- 单一政治人物表态不等于事实落地
- 若无进一步确认，容易只是一波短线噪音

## 12. 事后验证
- 验证市场是否真的按“战争降级”定价
- 验证哪一类资产最先切换风格

# 案例 3：OpenClaw 大火（热点事件）

## 1. 事件
- OpenClaw 成为热点话题，快速引发市场关注

## 2. 核心判断
- 这是典型的“叙事扩散型”热点事件
- 资金会优先流向最容易形成市场共识的映射标的

## 3. 逻辑链
热点爆发 → 市场寻找概念映射 → 龙头映射先涨 → 二级映射扩散 → 叙事退潮后分化

## 4. 资金流向
### 流入
- 直接映射股
- AI Agent / Agent Infra 概念资产
- 高传播性的 meme 映射币

### 扩散方向
- 生态合作方
- 工具链 / 基础设施相关标的

## 5. 跨市场映射标的
### 加密
- “龙虾” meme 币
- AI Agent / Agent Infra / 叙事映射币（后续持续补池）

### 港美股
- 腾讯
- MiniMax（若存在公开市场映射）
- 其他 OpenClaw / Agent 概念股

### 预测市场
- 若出现 adoption / AI agent 相关事件合约，可作为二级观察

## 6. 主交易标的
- 腾讯
- 龙虾 meme

## 7. 高弹性标的
- OpenClaw 概念小市值映射币
- AI Agent 小市值映射资产

## 8. 观察标的
- AI Agent 基础设施概念股/币
- 生态合作方
- 二级映射资产

## 9. 触发条件
- 热点继续扩散
- 官方 / 公司 / 机构继续绑定 OpenClaw 叙事
- 龙头放量上涨后出现板块扩散

## 10. 失效条件
- 热度快速退潮
- 龙头冲高回落，二级映射未扩散
- 市场不再给叙事溢价

## 11. 风险提示
- 热点映射交易常常最先涨的不是“最正宗”，而是“最易传播”
- 容易过热，需防一日游

## 12. 事后验证
- 验证一级映射与二级映射谁更有效
- 验证股市 / 加密 / 预测市场的传导顺序
