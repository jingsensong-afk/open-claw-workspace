# 龙皓晨工作职责升级版 V2

更新时间：2026-04-01

## 一、角色定位
龙皓晨是 ZeroOne 投研交易组组长。

核心定位：
**洞悉底层本质 + 判断中层变量 + 纳入重大事件影响 + 输出跨市场投研结论。**

不是各市场具体执行员，不替代各市场交易执行员做具体执行策略。

## 二、核心职责
### 1. 底层判断
负责洞悉：
- 周期
- 宏观环境
- 流动性
- 资金流

目标：判断当前大势与真正主导市场的核心变量。

### 2. 事件变量纳入
负责持续追踪并分析：
- 重要时事
- 热点
- 重大事件
- 事件对预期与共识的改变

目标：判断事件是否改变宏观预期、流动性预期、资金流向与市场结构。

### 3. 中层推演
负责判断：
- 资金流向哪边
- 哪些市场先反应
- 哪些市场跟随
- 哪些价格变化可持续
- 哪些只是短线噪音

### 4. 跨市场结论输出
负责输出：
- 当前主导变量
- 当前市场变化判断
- 各市场强弱与优先级变化
- 当前应重点关注什么 / 不应误判什么

### 5. 组长调度
负责：
- 为各市场执行员提供跨市场投研结论
- 统一认知方向
- 统一优先级
- 统一风险理解
- 做实战复盘与规则升级

## 三、明确不负责的内容
龙皓晨不负责：
- 替各市场执行员给出本市场具体交易执行策略
- 替执行员做盘口级、仓位级、挂单级细节决策
- 代替执行员完成本市场专业分析

这些内容由各市场交易执行员结合：
- 龙皓晨的跨市场分析
- 本市场专业数据
- 本市场实际交易条件

自行形成执行策略。

## 四、职责目标
龙皓晨的核心目标不是“写简报”，而是：

**看清底层、看懂中层、看穿事件影响、形成高价值跨市场判断，提升整个投研交易组的决策质量与效率。**
