# 热点异动机会报告示范稿（当前能力版）

## A类：主流可执行异动（主推标的）

- 标的：
  某热点异动币种 / 事件映射标的

- 一句话结论：
  有交易价值，但当前更适合快节奏跟踪与条件触发，不适合无确认重仓追高。

- 异动类型：
  事件驱动型放量异动

- 异动原因：
  热点事件催化 + 短线资金集中涌入 + 市场情绪快速扩散

- 当前阶段：
  异动启动后第一轮发酵阶段

- 是否值得做：
  值得看，但要挑位置和确认，不是无脑追

- 方向判断：
  短线偏多，回撤承接优于追高硬打

- 当前证据等级：中
- 是否受主报告关键缺口影响：是

- 主剧本：
  - 成立条件：异动后回踩不破关键支撑，并出现二次放量上冲
  - 继续有效条件：情绪不退潮、成交量不塌、主流市场不明显转弱
- 次剧本：
  - 触发条件：第一次拉升后进入横盘震荡，等待下一次催化
  - 升级为主剧本条件：放量突破前高 / 前压制区
- 反向剧本：
  - 触发条件：放量冲高后快速回落，跌破关键承接位且收不回

- 最优动作：
  只做确认后的低风险切入；若已大幅脱离成本区，优先观望，等二次机会。

- 关键压力：
  第一波拉升高点 / 前高密集成交区

- 关键支撑：
  异动起涨位 / 回踩承接位

- 失效条件：
  热点退潮，量能快速衰减，且价格跌破关键支撑后无法收回。

## B类：事件 / 叙事驱动观察名单（非主推也必须纳入视野）

- 标的：
  RAVE
- 是否纳入主推：
  否
- 未纳入主推原因：
  当前更偏事件传播驱动，短线弹性强，但流动性、持续性与滑点容错不如 A类主推标的。
- 当前异动核心：
  社区传播加速 + 事件叙事扩散，带来短线资金集中冲击。
- 当前阶段：
  第一轮情绪扩散阶段
- 观察结论：
  观望为主，等二次确认
- 若后续转强，升级为主推需满足：
  放量后不回吐、承接带清晰、主流市场不转弱、可给出明确关键位与失效条件。
- 当前主要风险：
  情绪退潮快、插针重、滑点大、持续性不足。
