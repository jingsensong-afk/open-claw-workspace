# Dangerous Manual Only

这里是主网危险脚本区。

本目录内脚本可能触发真实交易、真实账户变更、真实资金风险或其他不可逆外部动作。

## 硬边界
- 禁止 cron 调用。
- 禁止 `refresh_market_os_full_chain.sh` 调用。
- 禁止任何自动主链、自动刷新链、恢复链、分发链调用。
- 只能由人工明确理解风险后手动运行。
- 默认不建议运行。

## 运行前必须确认
- 当前脚本是否连接主网。
- 是否会下单、平仓、撤单、调杠杆或修改账户状态。
- 是否使用真实 API key / secret。
- 是否已有明确人工确认。
- 是否能接受可能产生的真实损失。

若不能完整确认以上问题，不要运行。
