# 王林｜Binance Square 指定账号抓取标准

更新时间：2026-04-24

## 目标
只抓对候选池有价值的 Binance Square 账号内容，降低噪音，提高信息流可信度。

## 一、优先抓取的账号类型
1. 实盘交易型账号
2. 高频更新型账号
3. 热点/新币/小币挖掘型账号
4. 带实盘交易组件 / 看涨看跌标签的账号
5. 有稳定互动量的账号

## 二、当前首批适合纳入的账号口径
### A类：实盘高频型
- 有持续发帖
- 明确涉及交易标的
- 能看到实盘/持仓/盈亏组件

### B类：热点放大型
- 对新币 / 上线币 / 热点赛道反应快
- 经常在广场形成高互动帖子

### C类：叙事挖掘型
- 对机构标 / 主题赛道 / 热点事件映射币敏感

## 三、什么帖子算有效
以下至少满足 1 项：
1. 明确提及具体币种
2. 明确表达看多 / 看空 / 强弱判断
3. 带实盘交易组件 / 仓位状态
4. 带高互动量（点赞/评论/浏览明显高）
5. 明确属于新币 / 上线 / 热点催化讨论

## 四、什么帖子不入池
1. 纯鸡汤
2. 无币种、无方向、无叙事的空话
3. 纯晒收益但没有标的/逻辑
4. 明显低质量转述
5. 与交易无关的闲聊

## 五、入池信号最小字段
- symbol
- narrative
- sentiment
- engagement_score
- source_account
- source_type=binance_square_account
- timestamp

## 六、默认处理规则
1. 先进入 watch pool
2. 多来源共振后再升 candidate
3. 若同时有交易所榜单 / X / Square 共振，优先升 execution 候选
