# 王林｜AI交易OS 真实小额执行 / API 接入准备规则

## 目标
在不直接上实盘的前提下，为后续真实小额执行与 API 接入做好最小可落地准备。

## 默认原则
1. 不在聊天中裸发高权限敏感信息。
2. 不开提现权限。
3. 优先测试账户 / 子账户。
4. 先做接入清单、字段清单、风控清单、最小执行白名单，再进入真实接通。


## Binance合约按U执行补充（2026-04-27新增）
- 系统执行意图层统一按名义价值（U）表达，不要求人工先决定币种数量。
- 例如：保证金 10U、杠杆 10X，对应目标名义价值 100U。
- 若底层交易所公开 API 不直接支持按U下单，则执行器必须在内部自动完成 `目标名义价值(U) -> 合法quantity` 的换算，并满足交易所 stepSize、quantityPrecision、minQty、minNotional 等规则。
- 这层换算属于执行器内部实现细节，不作为前台主逻辑暴露。
