# 新闻 / 宏观触发响应模块 V1

更新时间：2026-03-14

## 一、目标
建立一套用于识别和验证市场突发行情触发源的响应模块，重点解决：
- 行情突变时，能否快速判断是否有新闻/宏观事件触发
- 是技术面自发变盘，还是外部消息驱动
- 能否把宏观数据、官员讲话、地缘事件、政策消息与盘面变化快速对齐

## 二、核心问题
1. 当前行情变化是否有新闻触发？
2. 触发源属于：
- 宏观数据
- 官员讲话
- 地缘突发
- 政策消息
- 公司/行业消息
3. 触发后市场是否真的在定价？
4. 是短期噪音，还是可持续事件驱动？

## 三、当前可用能力
1. `opennews-mcp`
- 用于新闻/快讯/热点事件扫描

2. `opentwitter-mcp`
- 用于 X / Twitter 舆情扩散验证

3. `summarize`
- 用于快速提炼长文 / 链接

4. `Agent Browser`
- 用于抓取特殊网页与补位验证

## 四、分析流程
### 第一步：盘面异常识别
- 价格突然异动
- 量能放大
- 多资产联动异常

### 第二步：新闻触发验证
- 是否存在同时间段重大新闻/快讯
- 是否存在地缘/政策/宏观数据发布

### 第三步：舆情扩散验证
- 是否有 KOL / 主流账号迅速传播
- 情绪是否快速放大

### 第四步：市场是否相信
- 看价格、量能、跨市场联动是否确认
- 判断是噪音还是进入定价阶段

## 五、输出模板
1. 当前异动
2. 可能触发源
3. 新闻验证结果
4. 舆情验证结果
5. 市场是否定价
6. 当前判断：
- 消息驱动
- 技术性变盘
- 二者叠加
- 暂无明确触发源
7. 风险提示

## 六、使用原则
1. 不把单条快讯直接等同于有效触发源
2. 必须把新闻与价格/量能/联动一起验证
3. 先看触发，再看市场是否相信
4. 重点用于解释“为什么现在突然动了”

## 七、当前结论
该模块将作为 ZeroOne 的实盘触发判断层使用，尤其适用于：
- BTC / 高波动资产突然拉升或跳水
- 黄金 / 原油 / 美元 / 美股同步异动
- 地缘、政策、宏观数据驱动的跨市场机会识别
