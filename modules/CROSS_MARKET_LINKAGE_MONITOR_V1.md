# 跨市场联动监控模块 V1

更新时间：2026-03-14

## 一、目标
建立一套跨市场联动监控能力，用于在实盘中更快判断：
- 当前是谁在主导盘面
- 是 crypto 自身结构在驱动，还是宏观变量在驱动
- BTC 的上涨/下跌是否得到黄金、美元、美债、原油、美股的验证
- 当前属于哪一种市场模式

## 二、监控对象
1. 加密
- BTC
- ETH
- SOL（高 beta 代理）

2. 贵金属 / 商品
- 黄金（XAU / GC）
- 原油（WTI / Brent）

3. 宏观变量
- 美元指数（DXY）
- 美债收益率（10Y / 实际利率代理）

4. 风险资产
- 美股指数（QQQ / SPY）
- 半导体 / AI 主线（可选）

## 三、要回答的核心问题
1. 当前盘面主导变量是谁？
2. BTC 的波动是内部资金驱动，还是外部宏观变量驱动？
3. 黄金、美元、美债、原油是否共振？
4. 当前市场属于哪种模式：
- risk-on
- risk-off
- 去杠杆
- 美元走强压制
- 地缘溢价驱动
- 产业主题单独驱动

## 四、联动判断框架
### A. risk-on 模式
- BTC 涨
- ETH / SOL 同步更强
- QQQ / 风险资产偏强
- 美元不明显走强
- 黄金未必强

### B. risk-off 模式
- BTC 跌
- QQQ / 风险资产跌
- 美元偏强
- 黄金偏强或抗跌
- 原油通常偏弱（除非地缘驱动）

### C. 去杠杆模式
- BTC 跌
- 黄金也跌
- 风险资产跌
- 美元走强
- 说明市场先卖流动性好的资产回收现金

### D. 美元走强压制模式
- DXY 明显上行
- 黄金承压
- 美债收益率偏强或实际利率抬升
- BTC / 风险资产被压制，但不一定同步崩

### E. 地缘溢价模式
- 原油涨
- 黄金涨
- 风险资产承压
- 预测市场 / 地缘事件同步升温

## 五、输出模板
1. 当前各市场快照
- BTC / ETH / SOL
- 黄金
- 原油
- DXY
- 美债收益率
- QQQ / SPY

2. 当前联动关系
3. 当前主导变量
4. 当前市场模式判定
5. 对 BTC / 风险资产的含义
6. 风险提示

## 六、使用原则
1. 不单看 BTC 一个资产
2. 必须把加密、宏观、商品、权益至少四类变量拉到同一时间轴看
3. 优先回答“当前盘面是谁在主导”
4. 不先讲原理，先给实盘联动结论

## 七、当前结论
该模块将作为 ZeroOne 在实盘分析中的通用联动层使用，尤其适用于：
- BTC / 高波动资产分析
- 黄金 / 原油 / 美元联动分析
- 宏观变量驱动的跨市场机会识别
- 判断一段行情到底是 risk-on、risk-off、去杠杆，还是单一主题驱动
