# 市场是否相信自动验证链 V1

## 目标
建立一个可重复执行的验证链，用来判断：
**某条信息/事件，市场是否真的相信，并已经开始定价。**

## 核心原则
不能只看新闻标题或口头表态，必须交叉验证：
1. 情绪是否扩散
2. 价格是否响应
3. 成交量是否跟随
4. 预测市场是否同步变化
5. 衍生品（Funding / OI）是否确认

## 当前接入能力
1. `market-sentiment-pulse`
- 作用：做基础情绪辅助判断（轻量）

2. `polymarket-api`
- 作用：查预测市场是否有相关事件市场，以及概率是否变化

3. `hyperliquid`
- 作用：看 BTC 等核心资产的 funding / OI / volume / quote

4. 交易所行情（Binance）
- 作用：看价格与 24h 变化、成交量

## 验证链流程
### 第一步：信息分类
- 这条信息属于：
  - 重大事件
  - 热点事件
  - 口头表态
  - 实质进展

### 第二步：情绪验证
- 用 `market-sentiment-pulse` 做轻量辅助
- 结论只能作为参考，不能单独定性

### 第三步：价格验证
- 看核心映射资产是否出现一致方向变化
- 例如：
  - 风险偏好修复 → BTC / QQQ / 科技偏强，黄金/原油偏弱
  - 避险升温 → 黄金/原油偏强，风险资产偏弱

### 第四步：成交量验证
- 看价格变化是否伴随明显成交量
- 无量上涨/下跌，可信度较低

### 第五步：衍生品验证
- 用 `hyperliquid` / `funding-rate-scanner` 看：
  - Funding 是否变化
  - OI 是否抬升/回落
  - 是否出现拥挤方向

### 第六步：预测市场验证
- 用 `polymarket-api` 查相关事件市场
- 若预测市场概率同步变化，说明“市场相信”的证据更强

## 输出模板
1. 事件
2. 初步判断
3. 情绪验证
4. 价格验证
5. 成交量验证
6. 衍生品验证
7. 预测市场验证
8. 最终结论：
   - 市场未相信
   - 市场部分相信
   - 市场已开始定价

## 当前样例（特朗普称战争将很快结束）
### 情绪验证
- `market-sentiment-pulse`：Neutral
- 说明：当前轻量情绪工具没有给出明显情绪偏移

### 价格验证
- BTC（Binance 24h）：-1.73%
- ETH（Binance 24h）：-1.15%
- Hyperliquid BTC 24h：-1.56%
- 说明：风险资产并未体现“明显 risk-on 修复”

### 衍生品验证
- Hyperliquid BTC funding：-0.0008%
- OI：26,969.22
- 说明：空头略拥挤，但并未出现强烈单边再定价

### 预测市场验证
- Polymarket 搜索 `war ceasefire`：无直接命中
- 说明：当前未拿到足够有效的预测市场侧确认

### 当前样例结论
- 这条信息更像“前置信号 / 口头表态”
- **市场尚未明显按“战争将快速结束”去强定价**
- 当前结论：**市场部分关注，但未形成强一致相信**

## 结论分级
### A. 市场未相信
- 情绪无扩散
- 价格无响应
- 量能不跟随
- 预测市场无变化

### B. 市场部分相信
- 有部分资产响应
- 但未形成跨市场共振
- 量价/衍生品/预测市场确认不完整

### C. 市场已开始定价
- 跨市场资产同步响应
- 量能放大
- 衍生品方向确认
- 预测市场概率同步变化

## 当前限制
- `market-sentiment-pulse` 仍是轻量版，不足以独立承担情绪验证
- `polymarket-api` 的事件检索要依赖市场命名匹配
- 暂无 Coinglass / 更强情绪面数据源
- 当前版本适合作为 V1 验证链，不是最终版
