# 短周期拐点预警规则 V1

更新时间：2026-03-14

## 一、目标
建立适用于 BTC 及高波动资产的短周期拐点预警规则，用于识别：
- 上冲后的加速延续
- 高位转弱后的快速回撤
- 假突破 / 假跌破
- 多空连锁踩踏的早期信号

## 二、输入要素
1. 价格结构
- 关键压力位 / 支撑位
- 前高 / 前低
- VWAP / 日内中枢

2. OI
- 绝对值变化
- 短时变化速度

3. Funding
- 当前方向
- 是否极端
- 是否快速反转

4. Volume
- 是否显著放大
- 是否配合突破/跌破

5. Book / 流动性
- 承接是否抽离
- 挂单真空区是否出现

## 三、预警规则
### A. 向上延续预警
满足以下 3 条及以上：
1. 价格逼近/突破关键压力位
2. Funding 仍负或仅微正
3. OI 抬升
4. Volume 放大
5. 上方挂单真空明显

#### 含义
- 空头仍拥挤
- 突破后容易继续挤空
- 可判断为“向上加速风险增加”

---

### B. 向下踩踏预警
满足以下 3 条及以上：
1. 跌破关键承接位 / 前低
2. OI 快速回落
3. 下方盘口抽单 / 买盘变薄
4. Volume 放大
5. Funding 高位钝化后转弱，或价格跌而 funding 未有效修复

#### 含义
- 多头开始止损
- 价格容易进入连锁回撤
- 可判断为“向下加速风险增加”

---

### C. 假突破识别
满足以下 2 条及以上：
1. 价格突破关键位但未放量
2. OI 未明显抬升
3. Funding 已明显过热
4. 很快跌回关键位内侧

#### 含义
- 更像诱多
- 不适合追高

---

### D. 假跌破识别
满足以下 2 条及以上：
1. 跌破关键位但未放量
2. OI 未明显下滑
3. Funding 极端负值开始回归
4. 很快收回关键位上方

#### 含义
- 更像洗盘
- 不宜在极弱点追空

## 四、输出模板
1. 当前关键位
2. OI 状态
3. Funding 状态
4. Volume 状态
5. Book / 流动性状态
6. 当前属于：
- 向上延续预警
- 向下踩踏预警
- 假突破
- 假跌破
- 中性等待
7. 风险提示
8. 失效条件

## 五、使用原则
1. 不单看价格，必须至少结合 OI / funding / volume 中的两项
2. 不把单次假动作直接定义成趋势反转
3. 先识别预警，再给交易判断
4. 适合作为高波动资产的辅助预警层，不单独替代完整投研分析
