# 投研交易小组｜执行协作协议

更新时间：2026-03-29

## 适用范围
用于投研交易小组在加密执行工作中的共享协作。

## 共享共识
1. 执行分析采用统一链路：底层环境 → 周期结构 → 资金行为 → 技术触发 → 剧本推演 → 执行动作
2. 报告前台输出以结论、周期结构、剧本、动作、关键位、失效条件为核心
3. 后台数据源可多元，但前台输出必须简洁、精准、可执行
4. 执行日志必须记录：原计划、触发原因、实际执行、偏差原因、下一步动作
5. 日终复盘必须产出：做对、做错、错过、偏差、规则保留/修改
6. 常规事项可自主处理；风险上限变化、方向变化、突发变化、重大机会必须快速上报

## 当前共享数据源优先级
1. Binance USDⓈ-M
2. OKX
3. Hyperliquid
4. Coinbase
5. AiCoin

## 共享目标
- 提高执行一致性
- 降低信息表达噪音
- 保证报告可执行、可复盘、可沉淀
